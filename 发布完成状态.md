## 🚀 用户使用流程

### 方式 1: 下载 Release 包 (推荐)
1. 访问 GitHub Releases 页面
2. 下载 `patent-hub-v0.1.0-windows-x86_64.zip`
3. 解压到任意目录
4. 配置 .env 文件
5. 双击 start.bat 启动
6. 浏览器访问 http://localhost:3000

### 方式 2: 从源码构建
```bash
git clone https://github.com/jsshwqz/patent-hub.git
cd patent-hub
cargo build --release
```
