## 使用流程

### 1. 下载
从 GitHub Releases 下载 ZIP 包

### 2. 解压
解压到任意目录

### 3. 配置
```bash
copy .env.example .env
# 编辑 .env 文件，填入 API 密钥
```

### 4. 启动
```bash
# 方式 1: 双击 start.bat
# 方式 2: 命令行运行
patent-hub.exe
```

### 5. 访问
浏览器打开: http://localhost:3000
