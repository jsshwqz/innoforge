## 目标 / Goal

让 OA 答复功能真正能用在实际专利审查意见答复中。核心问题：AI 没有真正"读懂"文档就写回复。

## 待解决问题 / Remaining Issues

### P0 — 首次启动体验
- [ ] `start.bat` 首次运行需下载依赖+编译（5-10分钟），用户可能误以为卡死
- [ ] 如果没有预先编译好的 release 二进制，双击 start.bat 会先显示 cmd 窗口然后长时间无反馈

### P1 — OA 页面用户体验
- [ ] 分析耗时较长（30-90秒），没有进度提示
- [ ] AI 输出格式虽然结构化，但前端未做解析后验证（用户看不出 AI 有没有正确理解文档）
- [ ] 特征对比表是纯文本表格，不支持交互式勾选确认

### P2 — 长期规划
- [ ] 切换 AI 服务商到 DeepSeek 后需要用户手动确认 .env 配置
- [ ] OA 分析不支持流式（SSE），用户等待时看不到中间结果
- [ ] 无 .docx 格式导出
