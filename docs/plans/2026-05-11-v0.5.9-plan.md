> 基于 v0.5.8 发布后的遗留项和未来方向

## P0 — 发布收尾（优先级最高）

| 任务 | 说明 | 预估 |
|------|------|------|
| 清理远程 worktree 分支 | `git push origin --delete claude/lucid-engelbart-8f807d` | 1 min |
| 清理本地 worktree 目录 | `exit-worktree remove` 或手动删除 `.claude/worktrees/lucid-engelbart-8f807d/` | 1 min |
| Linux/macOS 安装包更新 | 在对应平台重新编译并上传到 Release，或配置 CI 自动构建 | 需要 CI |
| Gitee Release 同步 | Gitee 上如需发布说明，需手动编辑同步 | 5 min |

## P1 — 短期改进

### 1. 搜索质量提升
- SerpAPI 中文搜索精度调优（目前中文 query 有时返回不相关结果）
- 本地 FTS5 搜索结果排序优化（结合位置权重 + 关键词密度）

### 2. AI 对话体验
- 对话历史跨设备同步（目前仅 localStorage，换设备丢失）
- 聊天消息分段加载（长对话目前全部渲染，可能卡顿）

### 3. 设置页改进
- SerpAPI 余额/用量显示（调用 SerpAPI 账户信息接口）
- DeepSeek 模型可选下拉（目前隐藏为固定 deepseek-chat）

## P2 — 中期规划

### 1. CI/CD 搭建
- GitHub Actions：每次 push 到 main 自动运行 `cargo test`
- 自动构建全平台 Release 安装包并上传到 Release

### 2. 创新验证增强
- Pipeline 步骤可视化（前端进度条细化到子步骤）
- 实验验证步骤（`ExperimentValidation`）的真实执行能力

### 3. 多语言支持
- 英文版 UI 完善（目前部分页面仍以中文为主）
- 国际化文案覆盖率提升
