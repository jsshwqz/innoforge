## P1 — 短期改进

### 1. 搜索质量提升
- SerpAPI 中文搜索精度调优（目前中文 query 有时返回不相关结果） ✅
- 本地 FTS5 搜索结果排序优化（结合位置权重 + 关键词密度） ✅

### 2. AI 对话体验
- 对话历史跨设备同步（目前仅 localStorage，换设备丢失） ✅
- 聊天消息分段加载（长对话目前全部渲染，可能卡顿）
- 创新推演“专家深度”增强（模型路由 + 结构化输出 + 评分闭环）

### 3. 设置页改进
- SerpAPI 余额/用量显示（调用 SerpAPI 账户信息接口） ✅
- DeepSeek 模型可选下拉（目前隐藏为固定 deepseek-chat） ✅

## P2 — 中期规划

### 1. CI/CD 搭建
- GitHub Actions：每次 push 到 main 自动运行 `cargo test`
- 自动构建全平台 Release 安装包并上传到 Release

### 2. 创新验证增强
- Pipeline 步骤可视化（前端进度条细化到子步骤）
- 实验验证步骤（`ExperimentValidation`）的真实执行能力

### 3. 多语言支持
- 英文版 UI 完善（目前部分页面仍以中文为主）
- 国际化文案覆盖率提升
