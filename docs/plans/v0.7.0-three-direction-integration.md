# v0.7.0 三合一能力集成计划

> 将 AI 角色预设（方向三）、MCP 工具扩展（方向一）、流水线第 16 步（方向二）同步集成到研创台。

## 一、目标

把对话中展示的专利分析能力（PDF 专利阅读、现有技术搜索、多专利对比分析、答复意见书生成）以三种形式深度集成到研创台代码中，能力**远超原有水平**：

1. **AI 对话角色预设** — 5 种角色（专利审查专家、OA 答复专家、权利要求分析师、发明人头脑风暴），侧边栏切换
2. **MCP 服务器扩展** — `patent_threat_assessment`、`patent_claim_chart`、`patent_multi_compare` 三个新工具
3. **流水线第 16 步** — `GenerateOaResponse`，整合前 15 步产出自动生成 OA 答复辅助分析

## 三、架构说明

### 3.1 角色预设系统

```
用户选择角色 → sideb ar dropdown → system_prompt + preset_mode → POST /api/ai/chat
                                                                          ↓
                                                                  effective_system_prompt()
                                                                          ↓
                                                              resolved prompt → AiClient.chat()
```

预设定义在 `patent.rs` 的 `SYSTEM_PROMPT_PRESETS` 静态数组中，每个预设包含 `(key, label, prompt_text)`。

### 3.2 MCP 工具流

```
Claude Code (MCP 客户端) → stdio JSON-RPC → innoforge-mcp → HTTP POST → InnoForge (port 3000)
                                                                    ↓
                                                              AI analysis → text response
```

新增工具依赖已有的 `/api/ai/threat-assessment`、`/api/ai/claim-chart`、`/api/ai/compare-matrix` 后端端点。

### 3.3 流水线第 16 步

```
PipelineStep::Finalize (step 15) → next() → GenerateOaResponse (step 16)
                                                 ↓
                                steps::oa_response::execute(ctx, &ai_client)
                                                 ↓
                          AI chat → ctx.oa_response = "5 部分结构化报告"
```

Step 16 是 LLM 类型步骤，同 AiDeepAnalysis 相同的执行模式。

## 四、能力对比

### 原有能力（v0.6.3）
- OA 分析：3 类型（一审/非正常/复审）× 3 深度（简要/中等/深度），独立页面
- MCP 工具：6 个（patent_search/patent_detail/patent_analyze/patent_compare/idea_validate/patent_chat）
- 流水线：15 步，无 OA 相关步骤
- AI 对话：单一通用角色

### 新增能力（v0.7.0）
- 角色预设：5 种专用角色，侧边栏一键切换
- MCP 工具：+3 个（威胁评估/权利要求对照表/多专利对比），共 9 个
- 流水线：16 步，第 16 步自动生成 OA 辅助分析
- 威胁评估：批量 X/Y/A 分类 + 组合威胁 + 答辩策略
- 权利要求对照：逐元素映射 + 等同分析 + 技术效果评估
