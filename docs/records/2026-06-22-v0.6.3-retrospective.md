## 关键教训

- **CLAUDE.md 必须先读再动手**：项目宪法规定了所有规范和禁忌，跳过这一步等于是盲干
- **前端问题必须浏览器验证**：Failed to fetch 可能是 JS 崩溃导致的（DOMPurify 失效），curl 测不出来
- **Windows 环境变量和 Git Bash 环境变量是两回事**：`HTTP_PROXY` 在 Windows 系统中，reqwest 在 Rust 中自动读取，但 `unset` 在 Git Bash 里不影响 Windows 进程
- **系统代理的影响比想象中大**：Clash/v2ray 等代理工具设置的 `HTTP_PROXY` 会被所有 Rust HTTP 客户端继承
- **推理模型的 API 超时需要单独考虑**：deepseek-reasoner/v4-pro 处理长 prompt 需要 60-180 秒，45 秒的默认超时不够
- **.bat 文件中的 PATH 要自己处理**：不要假设 cargo 在 Windows 系统 PATH 中，`set "PATH=%USERPROFILE%\.cargo\bin;%PATH%"` 是最低保障
