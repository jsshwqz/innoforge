🚀 推送到 GitHub 的步骤

第一步：创建 GitHub 仓库
   1. 访问 https://github.com/new
   2. 仓库名称: patent-hub
   3. 描述: 🔍 开源专利检索系统 | Open Source Patent Search System
   4. 类型: Public（公开）
   5. 不要勾选 "Initialize this repository with a README"
   6. 点击 "Create repository"

第二步：推送代码
   方式 1（推荐）：
      双击运行: push_to_github.bat
   
   方式 2（手动）：
      git remote add origin https://github.com/YOUR_USERNAME/patent-hub.git
      git branch -M main
      git push -u origin main

第三步：配置仓库
   1. 添加 Topics（标签）：
      rust, patent, search, ai, axum, sqlite, open-source
   
   2. 设置 About：
      🔍 开源专利检索与分析系统 - 支持在线搜索、AI分析、专利对比
   
   3. 发布 Release v0.1.0：
      - 点击 "Releases" > "Create a new release"
      - Tag: v0.1.0
      - Title: v0.1.0 - Initial Release
      - 详见 GITHUB_SETUP.md
