# 工具脚本

## 发布工具

- `创建Release-最简单.bat` - 一键创建 GitHub Release（推荐）
- `auto_create_release.ps1` - PowerShell 自动发布（需要 Token）
- `auto_create_release.sh` - Bash 自动发布（需要 Token）

## 构建工具

- `build.bat` - 构建可执行文件
- `打包发布.bat` - 打包成 ZIP

## 推送工具

- `push_to_github.bat` - 推送到 GitHub
- `push_to_github.sh` - 推送到 GitHub (Bash)

## 配置工具

- `国内用户配置.bat` - 一键配置国内环境

## Release 文档

- `RELEASE_DESCRIPTION.md` - Release 说明模板
- `RELEASE_NOTES_v0.1.0.md` - v0.1.0 版本说明
